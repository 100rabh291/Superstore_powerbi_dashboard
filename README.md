# Global Superstore Sales Performance Dashboard — Power BI

## 📊 Overview
An interactive Power BI dashboard analysing retail sales performance across regions, product categories, and time, built on the widely-used Sample Superstore dataset.

## 📁 Dataset
**Source:** Sample Superstore (global retail transactions dataset)
**Scope:** 9,994 transaction rows across 4 regions and 49 US states

## 🎯 Business Questions Answered
- Which region generates the highest sales and profit?
- Which product category performs best?
- Which states are most profitable?
- How do sales trend over time?

## 🛠️ Tools & Techniques
- Power BI Desktop
- DAX measures (Total Sales, Total Profit, Distinct Order Count)
- 5 interactive visuals: Clustered bar charts, donut chart, KPI cards, line chart
- Interactive Region slicer for dynamic filtering
- Top N filtering for state-level profit ranking

## 🔍 Key Findings
1. **West region** leads in both sales ($7.4L) and profit ($1.1L)
2. **Technology** is the top-performing product category by sales
3. Central region shows high sales but disproportionately low profit — worth investigating
4. Dashboard supports real-time filtering by region for deeper drill-down analysis

## 📈 Dashboard Features
- 3 KPI cards: Total Sales, Total Profit, Total Orders
- Regional sales comparison (bar chart)
- Category breakdown (donut chart)
- Top 10 states by profit (bar chart)
- Interactive region slicer affecting all visuals simultaneously
